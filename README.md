# Immedis_AQA
* `./gradlew build` - Run tests
* `./gradlew allureServe` - Report will be generated into temp folder. Web server with results will start.
* `./gradlew allureReport`- Report will be generated t–æ directory: `build/reports/allure-report/allureReport/index.html`
